# ⚡ 英超百科 - 终极球迷挑战赛 (Premier League Quiz)

欢迎来到 **英超百科** 的开源项目！
这是一个基于 **英超官方历史数据** 驱动的网页小游戏。
我们希望借助社区的力量，打造 **最全、最硬核、最有趣** 的英超题库。

👉 在线试玩：[https://plmoneytalk.netlify.app](https://plmoneytalk.netlify.app)

---

## 🎮 游戏特色

* **海量题库**：涵盖历史纪录、冷门数据、球员成就等。
* **随机出题**：每次挑战随机抽取 10 道题，杜绝刷题背板。
* **评级系统**：从 *业余观众* 到 *英超传奇* 的趣味段位。

## 🤝 如何贡献题目（一起完善题库！）

觉得现在的题不够变态？
发现了新的纪录？
欢迎你一起扩展题库！

---

### **方法 1：直接修改代码（推荐）**

1. 下载上方文件列表中的 **index.html**
2. 找到代码中的
   `const questionPool = [`（大约在 **150 行** 左右）
3. 参照下面格式添加你的题目：

```js
{
  question: "在这里写下您的问题？",
  options: ["选项A", "选项B", "选项C（正确答案）", "选项D"],
  correctAnswer: "选项C（正确答案）",
  explanation: "在这里写下简单的解析，例如：该纪录创造于 2024 年。"
},
```

4. 修改完成后，点击页面底部的 **“Propose changes”** 按钮提交。

---

### **方法 2：提交 Issue（适合不会写代码的朋友）**

点击仓库顶部的 **Issues → New Issue**
把你的题目和正确答案写进去，我们会定期整理加入题库。
